# SmartLockManagement
SmartLockManagement is a smart doors' lock management cloud platform.

## build

steps

1. mkdir build
2. cd build
3. cmake ..
4. make
5. cd ../bin/
6. ./sam
7. visit local IP: **127.0.0.1:8888**(or other ports defined by variable *s_http_port*)
