# SmartLockManagement
SmartLockManagement is a smart doors' lock management cloud platform.

# build

```cmake
1. mkdir build
2. cd build
3. cmake ..
4. make
5. cd ../bin/
6. ./sam
7. 访问本地IP: 127.0.0.1:8888(或者自定义端口)
```
